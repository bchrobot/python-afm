# Abdul for Michigan

Helper scripts for the Abdul for Michigan campaign.


# Installation

Simply run:

    
```sh
$ git clone https://github.com/bchrobot/python-afm.git
$ cd python-afm
$ cp .env-dist .env
$ vi .env
# Update Twilio variables

$ pip install .
```


# Usage

To use it:

```sh
$ afm --help
```
